手把手教你制作个人网页 点击查看视频教程

第一次尝试做个人网站，快来跟着我一起学吧~
Step1：百度下载一个Bootstrap框架网页模板、Sublime Text编辑器。按照各人需求改变模板内的信息。
Step2：注册一个新浪sae账号、Github账号。
Step3：下载安装Git、Github客户端。
Step4：在新浪sae里新建一个应用，运行环境为PHP5.3空应用。。
Step5：打开Git客户端，进入模板所在文件夹，在你应用的git代码目录里，添加一个新的git远程仓库 sae
$ git remote add sae https://git.sinacloud.com/sheep3
编辑代码并将代码部署到 `sae` 的版本1。
$ git add .
$ git commit -am "make it better"
$ git push sae master:1
之后输入自己用户名和密码,是sae的安全邮箱和安全密码，不是微博账号和微博密码！
Step6：部署成功后，在新浪云应用中该应用的链接处即可进入自己的网页。
